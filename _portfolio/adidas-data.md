**Objective:** Designed and implemented a centralized data pipeline to automate the integration of global supplier manufacturing data into Adidas’s cloud infrastructure.

### The "Linker" Achievement
Managed the end-to-end integration of fragmented data sources from multiple departments, overcoming "Communication Inefficiency" and aligning stakeholder benefits through technical automation.

### Key Features
*   **Real-time Visibility:** Built a Power BI dashboard that refreshes automatically upon new data arrival via Power Automate.
*   **Data Pipeline:** Engineered the ingestion of Excel-based supplier data into a structured cloud environment.
*   **Strategic Planning:** Successfully reduced manual processing time by 90% and minimized human error in logistics reporting.

### Tech Stack
*   Power Platform (Automate, BI), Excel/VBA, Cloud Data Ingestion.
